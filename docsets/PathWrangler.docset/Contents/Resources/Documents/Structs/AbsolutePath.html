AbsolutePath

public struct AbsolutePath : _PathProtocol

Represents an absolute path.

  • resolve(resolveSymlinks:)

    Resolves (simplifies) the receiver, optionally including symlinks. By default this tries to resolve e.g. current directory (.) or parent directory (..) references. If resolveSymlinks is set to true, it also attempts to resolve symlinks. This requires the path to exist.

public mutating func resolve(resolveSymlinks: Bool = false)

    Parameters

    resolveSymlinks

    Whether or not symlinks should be tried to resolve. This will only work if the path exists on disk.
  • resolved(resolveSymlinks:)

    Returns a new path that has been resolved (simplified), optionally including symlinks. By default this tries to resolve e.g. current directory (.) or parent directory (..) references in a copy of the receiver. If resolveSymlinks is set to true, it also attempts to resolve symlinks. This requires the path to exist.
